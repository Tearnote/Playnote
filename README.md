# Playnote

The next generation [BMS](https://en.wikipedia.org/wiki/Be-Music_Source) player for Windows and Linux, in daily development. Not yet ready for general use.

![screenshot.png](screenshot.png)

## Development

Join the [Discord server](https://discord.gg/fnUpmAAjb9) to track and discuss development.

Check out the [Dev Blog](https://tear.moe/tag/playnote/) for progress reports and other various write-ups.

See the [Wiki](https://github.com/Tearnote/Playnote/wiki) for usage, building and documentation, and consult the [Roadmap](https://github.com/users/Tearnote/projects/4/views/1) for current development status.

## Builds

Find the latest Windows and Linux test builds in [CI artifacts](https://github.com/Tearnote/Playnote/actions).
